> A no more terminal-based **International Restaurant Management System** built with Python and MySQL.  
> Automates order management, menu control, and customer feedback — reducing manual work by ~70%.

<i><a href="https://foodportal-amber.vercel.app">Link to web portal</a> </i>

---

## 📌 Features

### 👨‍💼 Admin Panel
- Add, update, and delete menu items
- View all orders in a clean formatted table
- Password-protected access

### 🧑‍🍳 Customer Portal
- Browse a live menu of **62 dishes across 12 world cuisines**
- Place orders with bill generation
- View and cancel orders by phone number
- Submit feedback with star ratings

### 🗄️ Database
- 3 relational tables: `item`, `orders`, `feedback`
- Foreign key constraints and indexed lookups
- UTF-8 support for international dish names
